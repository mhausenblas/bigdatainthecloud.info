Big Data in the Cloud

The Q & A site for Big Data processing in cloud computing environments.

Big Data processing offerings in the cloud, such as Hadoop-as-a-Service (HaaS), are increasingly becoming available and popular. At the same time, developers, devops, administrators, and business users alike seek guidance concerning good practices, gotchas and tips using these offerings. This site aims to provide orientation and a discussion platform for the topic 'Big Data processing in the cloud'.
